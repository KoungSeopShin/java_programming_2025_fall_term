# 지뢰찾기 게임

## 문제 설명
- `n×m`의 2차원 배열을 만들고 `k`개의 지뢰를 임의의 위치에 삽입한 뒤, 인접한 지뢰의 수를 표시한 배열을 출력하는 프로그램.
- 다차원 배열 학습을 위함임.

## 코드 구조 
- `Minesweeper` 한 개의 클래스, `main` 메소드 한 개의 메소드로 구성됨.
- 2차원 배열 생성 후, `while`문에서 지뢰를 무작위 배치함.
- 그 이후, 이중 `for`문 안에 다중 `if`문으로 `-1` 주변 8개의 값에 `+1`을 함.
- 마지막으로 2중 `for`문으로 배열 출력.

## 코드 실행과정
1. `Scanner` 객체를 생성하기 위한 `scanner` 레퍼런스를 만듦.
2. 배열의 크기인 `int n, m`을 입력 받음.
3. 지뢰의 개수인 `k`도 입력받음, 만약 `n*m`의 값보다 크다면 실행 종료.
4. 2차원 배열 `intArray`와 `Random` 객체 레퍼런스 `random` 생성.
5. `while`문에서 `(l <= k)` 조건을 위한 `int l`을 생성함.
6. `while`문 안에서 `int r, d`에 `random.nextInt()` 함수로 `n, m` 사이의 값을 삽입,  
   `if`문에서 `intArray[r][d]`가 `-1`이 아니라면 `-1` 삽입 후 `l++` 함.
    ```java
    while (l <= k) {
        int r = random.nextInt(n);
        int d = random.nextInt(m);
        if (intArray[r][d] != -1) {
            intArray[r][d] = -1;
            l++;
        }
    }
    ```
7. `while`문을 빠져 나온 뒤, 이중 `for`문으로 1행 1열부터 검사 진행.
8. 최상위 `if`문에서 `-1`인지 아닌지 검사, 이후 가장 왼쪽 행 3칸, 같은 열 상하, 오른쪽 3칸의 조건문을 묶은 뒤, 각각의 칸에 대한 `if`문 실행.
    ```java
    for (int i = 0; i < intArray.length; i++) {
        for (int j = 0; j < intArray[i].length; j++) {
            if (intArray[i][j] == -1) {
                // 왼쪽 3칸
                if (i > 0) {
                    if (j > 0 && intArray[i-1][j-1] != -1) intArray[i-1][j-1] += 1;
                    if (intArray[i-1][j] != -1) intArray[i-1][j] += 1;
                    if (j < m-1 && intArray[i-1][j+1] != -1) intArray[i-1][j+1] += 1;
                }
                // 같은 열 상/하
                if (j > 0 && intArray[i][j-1] != -1) intArray[i][j-1] += 1;
                if (j < m-1 && intArray[i][j+1] != -1) intArray[i][j+1] += 1;

                // 오른쪽 3칸
                if (i < n-1) {
                    if (j > 0 && intArray[i+1][j-1] != -1) intArray[i+1][j-1] += 1;
                    if (intArray[i+1][j] != -1) intArray[i+1][j] += 1;
                    if (j < m-1 && intArray[i+1][j+1] != -1) intArray[i+1][j+1] += 1;
                }
            }
        }
    }
    ```
9. `for`문 빠져나온 뒤, 이중 `for`문으로 배열 출력, 해당 칸이 `-1`이면 공백문자 하나, 아니라면 두 개를 출력해서 가독성 높임.

## 느낀 점
- 교수님이 과제를 먼저 온라인으로 올려주시고 수업에서 과제에 대한 설명을 해주셨다.  
- 근데 난 이미 온라인에 올라오자마자 끝내서 교수님의 설명과 조금 다르게 개발되었다.  
- 교수님의 설명에서 가장 다른 점은, 교수님 방식은 모든 칸의 주변 8칸을 검사해서 `-1`이 있다면 해당 칸에 `+1`을 하는 것이었지만, 내 코드는 `-1`을 먼저 탐색한 뒤 그 주변의 칸에 `+1`을 하는 방식이다.  
- 내 코드가 더 효율적이라고 생각하지만 직관적인 건 교수님 방식인 것 같기도 하다.  
- 또한 교수님은 배열을 생성할 때, 그 값을 초기화하는 `for`문을 사용하는 것이 좋다고 하셨다. 난 그런 코드가 없다. 다음부턴 그렇게 해야겠다.
